Control of two pololu motors.

Pinout:
PC13 -> Internal LED.
PB08 and PB09 -> Timer 4 PWM CH3 (left motor) and CH4 (right motor).
PB12 and PB13 -> Motor driver control pins for left motor.
   L        L  -> brake low
   H        L  -> forward
   L        H  -> reverse
PA09 and PB10 -> Serial port Tx/Rx.
